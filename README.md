# ELIXIR TECHNOLOGY

Python + Kivy ile geliştirilmiş, Bluetooth Classic üzerinden Arduino sensör verilerini okuyan, 2D/3D yer altı görüntüleme ve gerçek zamanlı pointer modu içeren Android uygulaması.

## 📱 Özellikler

### Yer Altı Görüntüleme Modu
- Bluetooth Classic bağlantı
- 5x5 varsayılan grid (özelleştirilebilir)
- Zigzag veya paralel tarama
- Gerçek zamanlı 2D ısı haritası
- 3D yüzey görüntüleme (OpenGL)
- Derinlik hesaplama (manyetik gradyan formülü)
- CSV import/export
- Değer göster/gizle

### Pointer Modu
- Gerçek zamanlı analog değer gösterimi
- Pulse animasyonlu dairesel gösterge
- Değere göre renk değişimi (Mavi → Yeşil → Kırmızı)
- Sesli geribildirim (bip frekansı değere göre)
- Değer göstergesi

## 🛠️ Teknik Gereksinimler

- Python 3.8+
- Kivy 2.1.0
- Buildozer 1.4.0
- Android SDK (API 21+)
- Bluetooth Classic destekli Android cihaz
